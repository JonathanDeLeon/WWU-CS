# Java Quicksort

This quicksort algorithm was built in Java for *CPTR354 Compilers and Programming Languages* at Walla Walla University. 
The goal of the project was to implement the quicksort algorithm using a chosen programming language. 

## Setup
This project uses Apache Maven to compile and run the project. 
Here are a list of useful maven commands:

Compile project to byte code and output to the target directory
```
mvn compile
```
Execute main in QuickSort.java

**Note**: Project must be compiled first
```
mvn exec:java
```

Remove target directory
```
mvn clean
```
